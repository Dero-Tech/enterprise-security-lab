# SOC Analyst Academy

> A hands-on cybersecurity portfolio documenting my journey from healthcare and government service into Security Operations (SOC), Threat Detection, and Vulnerability Management.

## About This Project

This repository documents my practical cybersecurity training through a structured series of hands-on labs.

The goal is to gain real-world experience with Security Operations Center (SOC) workflows, threat hunting, vulnerability management, incident response, and security monitoring while building a professional portfolio for future employers.
