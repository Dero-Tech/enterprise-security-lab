# NorthStar Enterprise Identity & Security Lab

Enterprise-inspired Identity and Access Management platform built with FastAPI, SQLAlchemy, and Python.

## Current Features

- FastAPI REST API
- SQLAlchemy ORM
- SQLite persistence
- Pydantic validation
- OpenAPI documentation

## Planned Features

- JWT Authentication
- RBAC
- Audit Logging
- Joiner / Mover / Leaver
- Microsoft Entra ID concepts
- Active Directory integration
- Splunk monitoring

## Tech Stack

Python • FastAPI • SQLAlchemy • SQLite • Git • OpenAPI
